טוען נתונים מ-GitHub...

🔑 הגדרת סנכרון GitHub

כדי לשמור את הנתונים בענן ולסנכרן בין מכשירים, תצטרך טוקן גישה ל-GitHub.

  1. כנס ל: github.com/settings/tokens
  2. לחץ Generate new token
  3. תן שם (למשל: "Cocktails App")
  4. תחת Repository access — בחר Only select repositories ← eitan357/Cocktails
  5. תחת Permissions → Contents — בחר Read and write
  6. לחץ Generate token והעתק

הענף שבו האפליקציה מאוחסנת ב-GitHub (הענף שבחרת ב-GitHub Pages)
